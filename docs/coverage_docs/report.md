## Methodology

- **Technique:** Static analysis via Python's `ast` module. Each in-scope file is parsed and walked; `ast.get_docstring()` determines whether a function, method, or class has a docstring.
- **Visibility:** Symbols whose name starts with `_` are treated as private/optional and excluded from the public counts, **except `__init__`**, which is always counted as public.
- **Classes & methods:** Class bodies are walked so methods are counted; nested defs inside function bodies (local helpers) are not counted.
- **Scope:** `going_modular/going_modular/`, `helper_functions.py`, `screencropnet/`, `screennet/`, `tasks/`, `contrib/`, `label_pointer.py`, `preprocessing_data_loader.py`. **Excluded:** `tests/`, `.claude/`, `lint-configs-python/`, `going_modular/pytorch_ipynb/`, all `*.ipynb` notebooks, `scratch/`, and `.venv/`.
- **Numbers** are exact AST counts, not estimates.
